1. What is DOM and what is the purpose of DOM?

Document Object Model (DOM) plays an important role between web pages and programming languages.The main purpose of the DOM is to enable programs to access and manipulate web page content, structure, and style, and style of documents such as HTML, and change the web page structures as the developers want

3. What is event Bubble?

The Event bubbling in JavaScript is a type of event propagation. First of all, the event triggers the target element and consecutively triggers the parent element of the target element. It remains to target the parent until it reaches the top of the parent.

4.What is a Callback function and why will you use it?

A Callback function is a function that is to be executed after another function has finished execution. I will use the CallBack function for adding event handlers and getting input field values by passing parameters. So don't need to rewrite the full function again and again. That's why I will use a callback function.
